git相关

在主配置文件下面设置git链接的仓库文件名称
如果git没有相关配置，则会需要输入用户名和密码
否则直接使用git的config配置，来连接远程仓库，包括rsa免密认证

主题相关

主题等在文件夹theme下面，文件夹名称就是主题名
将可用的主题名在根文件夹下面的_config.yml中修改即可
